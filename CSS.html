CSS is a language used to alter the presentation of a html document. CSS stands for Casacding Style Sheet. This language allows you to change the color, position, and style of various text, images, objects, etc. that are on your webpage. Unlike HTML, you do not need opening and closing tags in CSS. In addition, whenn creating your webpage, you can either use one CSS document for all of your pages to make the layout uniform, or you can have separate CSS documents so that you may have different layouts on each page.
